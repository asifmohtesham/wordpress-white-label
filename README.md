# WordPress White Label

A clean, modular WordPress plugin that white-labels and customizes your WordPress site using the free features inspired by [Ultimate Dashboard](https://ultimatedashboard.io).

## Free Features Implemented

| Module | Feature |
|---|---|
| **Login Customizer** | Custom logo, background, form colors, button styles via WP Customizer |
| **Login Redirect** | Per-role redirect after login, block unauthorized `/wp-admin` access |
| **Branding** | Custom "Howdy" text, footer text, dashboard headline, hide WP version |
| **Dashboard Cleanup** | Remove Help tab, Screen Options tab, default dashboard widgets |
| **Custom Widgets** | Create icon & text widgets for the dashboard |
| **Welcome Panel** | Replace default WordPress welcome panel content |
| **Custom CSS** | Inject custom CSS into admin dashboard and login screen |
| **Admin Bar** | Remove admin bar for specific user roles |
| **Custom Admin Pages** | Create top-level or sub-menu admin pages with HTML/CSS |
| **Import / Export** | Export and import all plugin settings as JSON |

## Installation

1. Clone or download this repository.
2. Upload the `wordpress-white-label` folder to `/wp-content/plugins/`.
3. Activate the plugin in **Plugins → Installed Plugins**.
4. Navigate to **WP White Label** in the admin menu to configure.
